#### :hammer: ft_print_numbers.c

Print the Numbers - 0 to 9

ASCII <br>
- 48 = '0' <br>
- 57 = '9'
```c
void	ft_print_numbers(void)
{
	int	count;

	count = 48;
	while (count <= 57)
	{
		write(1, &count, 1);
		count++;
	}
}
```
